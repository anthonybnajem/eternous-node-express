# Requirements

This folder is the canonical place for project requirements, setup prerequisites, and integration notes.

## Contents

- `backend.md` - runtime and local development requirements
- `auth.md` - Firebase and legacy auth requirements
- `infrastructure.md` - MongoDB, Redis, Docker, and deployment requirements

## Current Scope

The app currently depends on:

- Node.js 18+
- MongoDB
- Redis
- Firebase project access for Firebase auth and notifications
- Stripe keys for payment flows
- SMTP credentials for email flows

If you want, I can expand this folder into separate requirement docs for each subsystem.
